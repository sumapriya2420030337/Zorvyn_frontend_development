# 💰 Zorvyn Fintech Dashboard

A modern fintech dashboard built using React + Vite with role-based UI and data visualization.

---

## 🚀 Features

- 🔐 Role-based access (Admin / Viewer)
- 📊 Interactive charts using Recharts
- 💾 LocalStorage persistence
- 📋 Transaction management
- 🎨 Glassmorphism UI design
- 📱 Fully responsive layout

---

## 🧱 Tech Stack

- React (Vite)
- Context API
- Recharts
- CSS (Glass UI)

---

## 📂 Project Structure

- components/
- pages/
- context/
- data/
